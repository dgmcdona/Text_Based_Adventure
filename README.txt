This is a text-based adventure game demonstrating the use of object oriented programming. One advanced feature is its use of the Timer and TimerTask classes. 

Setup:
  To ensure the proper functioning of this application, begin by removing existing class files.
    ex: 'rm -rf *.class'
  Next, assuming that you have the JDK and JRE properly installed, compile classes.
    ex: 'javac *.java'

To Run:
  'java Game'
